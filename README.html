---------------操作方法--------------

スタート : 選択したステージを遊びます

前のステージ : 後ろのステージを選択します

次のステージ : 次のステージを選択します

難易度 : そのステージの難しさを示しています

ステージ名称 : ステージの名前です

概要 : そのステージについての詳細が書いてあります
