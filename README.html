RollingSkyEvolution

ローリングスカイの二次創作ゲームがWEBで登場！超難易度の鬼畜ステージのクリアを目指せ！

このゲームはあくまで二次創作です。

メモリが少ない端末ではラグが発生する可能性が極めて高いです。ご注意ください

効果音が無いですが、IOSでのプレイが一番オススメです。(ラグが最も少ないため)

---------------操作方法--------------

スタート : 選択したステージを遊びます

スタートしたら 指 または マウス を左右に動かして操作します。100%に到達するとクリアになります。

鍵が掛かっていて出来ないステージは特定のステージをクリアすることでアンロックされます。

前のステージ : 後ろのステージを選択します

次のステージ : 次のステージを選択します

難易度 : そのステージの難しさを示しています

ステージ名称 : ステージの名前です

概要 : そのステージについての詳細が書いてあります
